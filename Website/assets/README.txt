assets/
-------
Place your static assets here:
  - images/       Profile photos, project screenshots, cover images
  - icons/        Favicon, app icons (favicon.ico, apple-touch-icon.png)
  - fonts/        Self-hosted web fonts (optional)
  - og/           Open Graph images for social sharing

Recommended image formats: WebP (primary), AVIF (next-gen), PNG/JPG (fallback)
Favicon: Place favicon.ico at the project root for best compatibility.
